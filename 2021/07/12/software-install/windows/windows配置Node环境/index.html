6. 出现问题

6.1 windows上安装nvm后选择node版本出现exit status 1…

问题：nvm install x.x.x 可以成功，但无法切换和使用

原因：我把 nvm 安装到了有空格的路径上（D:\Program Files），导致切换失败

解决：把 nvm 卸载重装到没有空格的路径上（例如：D:\nvm），即可解决问题
